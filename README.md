# TFAG2002 Final project portfolio

# Prosjektbeskrivelse
Dette prosjektet er en personlig portefølje for meg som UX-designer og frontend-utvikler. Nettsiden er bygget for å presentere min kompetanse, verdier og visuelle identitet på en måte som er tilgjengelig, troverdig og estetisk gjennomført. Den består av flere seksjoner: en introduksjon med verdikort, en oversikt over designprosjekter med tilhørende beskrivelser, en “Om meg”-side med refleksjoner og bakgrunn, samt en kontaktside med skjema og kontaktinformasjon. Målet er å gjøre det enkelt for potensielle arbeidsgivere og samarbeidspartnere å bli kjent med meg og ta kontakt. Porteføljen er responsiv og universelt utformet, og bygger på prinsipper for etisk og bærekraftig design.

# Oppsett og installasjon
Ingen avansert installasjon er nødvendig for å kjøre denne porteføljen lokalt. Prosjektet er bygget med HTML, CSS og JavaScript, og krever kun en moderne nettleser for å vises korrekt. Dersom du ønsker å gjøre endringer i kildekoden, anbefales det å ha Node.js installert for å kunne bruke eventuelle utviklingsverktøy som bundling eller live server. Ingen API-nøkler eller backend-integrasjoner er nødvendig i denne versjonen.

# Slik kjører du lokalt
Last ned prosjektmappen eller klon Git-repoet til din lokale maskin.

Åpne index.html direkte i nettleseren din.

Alternativt, hvis du bruker VS Code eller annen editor med live server:

Åpne prosjektmappen i editoren.

Høyreklikk på index.html og velg “Open with Live Server”.

Siden vil da åpnes automatisk i nettleseren.
